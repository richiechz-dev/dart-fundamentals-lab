A sample command-line application with an entrypoint in `bin/`, library code
in `lib/`, and example unit test in `test/`.

# Dart Learn Basics

Proyecto educativo con fundamentos del lenguaje Dart y notas personales del lenguaje, enfocado en preparación y actualización para Flutter.

## Estructura

- `bin/main.dart` - Entrada de ejemplo con `calculate()`
- `bin/01_basics/` - Introducción, clases, stdin y operadores lógicos
- `bin/02_funciones/` - Funciones y métodos
- `bin/03_colecciones/` - Listas, mapas e iterables
- `lib/models/` - Clases y objetos (por ejemplo, `ice_cream.dart`, `setters_and_getters.dart`)
- `lib/animals/` - Ejemplo de herencia (`animal.dart`)
- `lib/dart_learn_basics.dart` - Librería principal usada por `main.dart`

## Ejecutar

```sh
dart run bin/main.dart
